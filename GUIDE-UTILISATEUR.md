# 📖 Guide Utilisateur - Distribution CIDEX Bernin

## 🚀 Démarrage

1. Ouvrez `index.html` dans votre navigateur
2. Cliquez sur "Lancer l'Application"
3. Entrez un **code de session** (ex: "bernin2024")
4. Entrez votre **prénom**
5. Cliquez sur "Démarrer"

### Utilisation seul ou en équipe

L'application fonctionne de la même manière que vous soyez seul ou en équipe :

**Seul :**
- Créez votre propre code de session
- Vos données sont sauvegardées dans le cloud
- Vous pouvez reprendre votre distribution depuis n'importe quel appareil

**En équipe :**
- Partagez le même code de session avec votre équipe
- Vous verrez les autres membres actifs dans le panneau de droite
- Tout est synchronisé en temps réel

### Marquer un CIDEX comme distribué
1. Cliquez sur un marqueur **bleu** (à faire)
2. Il devient **vert** (distribué)
3. Cliquez à nouveau pour annuler

### Synchronisation
Tout est synchronisé en temps réel dans le cloud :
- ✅ États de distribution (fait/à faire)
- ✅ Ajout de CIDEX
- ✅ Déplacement de CIDEX
- ✅ Suppression de CIDEX
- ✅ Membres actifs (si en équipe)

## 🎯 Menu d'actions

Cliquez sur le bouton **☰** en bas à droite pour ouvrir le menu.

### ➕ Ajouter un CIDEX

**Quand l'utiliser :** Vous découvrez un CIDEX manquant sur le terrain

**Comment faire :**
1. Cliquez sur "➕ Ajouter un CIDEX"
2. Remplissez le nom (ex: "Chemin des Roses")
3. Remplissez l'adresse (optionnel)
4. Cliquez sur "Ajouter"
5. Un bandeau orange apparaît en haut
6. Cliquez sur la carte pour positionner le CIDEX
7. Le nouveau CIDEX apparaît en bleu

**Pour annuler :**
- Cliquez sur "Annuler" dans le bandeau avant de cliquer sur la carte

---

### 📍 Déplacer un CIDEX

**Quand l'utiliser :** Un CIDEX est mal positionné sur la carte

**Comment faire :**
1. Cliquez sur "📍 Déplacer un CIDEX"
2. Un bandeau orange apparaît en haut
3. Cliquez sur le marqueur à déplacer (il devient **orange**)
4. Cliquez sur la carte à la nouvelle position
5. Le marqueur se déplace et reprend sa couleur (bleu ou vert)
6. Répétez pour déplacer plusieurs CIDEX
7. Cliquez sur "💾 Sauvegarder" pour valider

**Pour annuler :**
- Cliquez sur "Annuler" dans le bandeau
- Tous les CIDEX retournent à leur position d'origine

**Important :**
- Les déplacements ne sont pas sauvegardés tant que vous ne cliquez pas sur "Sauvegarder"
- Les autres utilisateurs (si en équipe) voient les changements après sauvegarde

---

### 🗑️ Supprimer un CIDEX

**Quand l'utiliser :** Un CIDEX a été ajouté par erreur ou n'existe plus

**Comment faire :**
1. Cliquez sur "🗑️ Supprimer un CIDEX"
2. Un bandeau orange apparaît en haut
3. Cliquez sur le marqueur à supprimer
4. Une confirmation apparaît : "Voulez-vous vraiment supprimer..."
5. Cliquez sur "OK" pour confirmer
6. Le CIDEX disparaît immédiatement

**Pour annuler :**
- Cliquez sur "Annuler" dans le bandeau avant de cliquer sur un marqueur
- Ou cliquez sur "Annuler" dans la popup de confirmation

**Important :**
- La suppression est immédiate et définitive
- Le CIDEX disparaît pour tous les utilisateurs (si en équipe)

---

### 🔄 Tout réinitialiser

**Quand l'utiliser :** Nouvelle distribution, vous voulez remettre tous les CIDEX à "à faire"

**Comment faire :**
1. Cliquez sur "🔄 Tout réinitialiser"
2. Une confirmation apparaît
3. Cliquez sur "OK" pour confirmer
4. Tous les marqueurs redeviennent bleus (à faire)

**Important :**
- Ne supprime pas les CIDEX, seulement les états de distribution
- Réinitialise pour tous les utilisateurs (si en équipe)

## 📊 Panneau de statistiques

Toujours visible à droite (ou en bas sur mobile) :

- **Total CIDEX** : Nombre total de CIDEX
- **✅ Distribués** : Nombre de CIDEX distribués (verts)
- **⏳ Restants** : Nombre de CIDEX à faire (bleus)
- **Progression** : Barre de progression avec pourcentage
- **👥 Membres actifs** : Liste des membres connectés (si en équipe)

---

## 🎨 Couleurs des marqueurs

- **🔵 Bleu** : CIDEX à distribuer
- **🟢 Vert** : CIDEX distribué
- **🟠 Orange** : CIDEX sélectionné (mode déplacement)

## 💡 Astuces

### Pour une distribution efficace
1. Zoomez sur la zone que vous allez distribuer
2. Repérez les CIDEX bleus à proximité
3. Marquez-les en vert au fur et à mesure
4. Suivez la progression dans le panneau de droite

### En équipe
1. Divisez-vous les zones géographiques
2. Regardez les membres actifs pour savoir qui est où
3. Communiquez le code de session avant de commencer
4. Ajoutez les CIDEX manquants au fur et à mesure

### Sur mobile
1. Ajoutez l'application à l'écran d'accueil
2. Le panneau de statistiques est en bas
3. Utilisez deux doigts pour zoomer/dézoomer
4. Nécessite une connexion internet pour la synchronisation

---

## ❓ Problèmes fréquents

### La carte ne s'affiche pas
- Vérifiez votre connexion internet
- Rechargez la page (F5)

### Mes données ont disparu
- Vérifiez que vous utilisez le bon code de session
- Les données sont sauvegardées dans le cloud Firebase
- Vérifiez votre connexion internet

### Je ne vois pas les autres utilisateurs
- Vérifiez que vous utilisez le même code de session
- Vérifiez votre connexion internet
- Les utilisateurs inactifs depuis plus de 5 minutes disparaissent

### Un CIDEX ne se déplace pas
- Vérifiez que vous êtes en mode "Déplacer un CIDEX"
- Cliquez d'abord sur le marqueur (il devient orange)
- Puis cliquez sur la carte à la nouvelle position
