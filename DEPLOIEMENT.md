# 📦 Guide de Déploiement - CIDEX Bernin

## Option 1 : GitHub Pages (Recommandé ✅)

### Étape 1 : Créer un compte GitHub
1. Allez sur [github.com](https://github.com)
2. Cliquez sur "Sign up"
3. Créez votre compte gratuitement

### Étape 2 : Créer un repository
1. Cliquez sur le bouton "+" en haut à droite
2. Sélectionnez "New repository"
3. Nom : `cidex-bernin`
4. Cochez "Public"
5. Cliquez sur "Create repository"

### Étape 3 : Uploader les fichiers
1. Sur la page du repository, cliquez sur "uploading an existing file"
2. Glissez-déposez ces fichiers :
   - `index.html` (page d'accueil)
   - `app.html` (version solo)
   - `app-multi.html` (version équipe)
   - `README.md`
3. Cliquez sur "Commit changes"

### Étape 4 : Activer GitHub Pages
1. Allez dans "Settings" (en haut)
2. Dans le menu de gauche, cliquez sur "Pages"
3. Sous "Source", sélectionnez "main" branch
4. Cliquez sur "Save"
5. Attendez 1-2 minutes

### ✅ C'est prêt !
Votre app sera accessible sur :
```
https://VOTRE-NOM-UTILISATEUR.github.io/cidex-bernin/
```

## 📱 Utilisation sur mobile

Une fois déployé, vous pouvez :
1. Ouvrir l'URL sur votre téléphone
2. Ajouter à l'écran d'accueil (comme une app)
3. Utiliser hors ligne (les données sont sauvegardées)

---

## 🔧 Mise à jour de l'application

### Sur GitHub :
1. Allez sur votre repository
2. Cliquez sur le fichier à modifier
3. Cliquez sur l'icône crayon (Edit)
4. Faites vos modifications
5. Cliquez sur "Commit changes"
6. Les changements sont en ligne en 1 minute !

---

## 💡 Conseils

- **Partagez l'URL** avec votre équipe
- **Ajoutez un raccourci** sur l'écran d'accueil de votre téléphone
- **Testez d'abord** en mode solo avant d'utiliser le mode équipe
- **Sauvegardez** régulièrement vos données avec le bouton "Exporter"

---

## ❓ Besoin d'aide ?

Si vous rencontrez des problèmes :
1. Vérifiez que tous les fichiers sont bien uploadés
2. Attendez 2-3 minutes après l'activation de GitHub Pages
3. Essayez en navigation privée pour éviter les problèmes de cache
